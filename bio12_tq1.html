🔵 Carl Linnaeus (1700s) Father of taxonomy
Introduced binomial nomenclature
Two-part Latin naming system
🟠 Taxonomic Hierarchy Mnemonic: Dear King Philip Came Over For Good Soup
Domain → Kingdom → Phylum → Class → Order → Family → Genus → Species
🟢 Binomial Nomenclature Genus capitalized + species lowercase
Both italicized: Homo sapiens
Abbreviate genus after first use: H. sapiens
🔴 Phylogenetic Groups Monophyletic = ancestor + ALL descendants
Paraphyletic = ancestor + SOME descendants
Birds = reptiles (common ancestry)
