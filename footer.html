How to Integrate This Footer on Any Webpage

This page demonstrates the TkLesson footer and shows you how to integrate it into any webpage.

Integration Instructions

Add this code to your webpage where you want the footer to appear:

<!-- TkLesson Footer Container --> <div id="tklesson-footer"></div> <script> // Fetch and insert the footer fetch('https://yourdomain.com/path/to/footer.html') .then(response => response.text()) .then(html => { // Create a temporary container const tempContainer = document.createElement('div'); tempContainer.innerHTML = html; // Extract just the footer section const footerElement = tempContainer.querySelector('#tklesson-footer'); if (footerElement) { // Insert the footer into the placeholder document.getElementById('tklesson-footer').innerHTML = footerElement.innerHTML; } }) .catch(error => { console.error('Error loading footer:', error); }); </script>

Make sure to update the URL in the fetch() call to point to your actual footer.html location.

Important Notes

  • The footer styles are self-contained and won't affect your page's existing styles
  • No global CSS resets are applied, so your page layout won't be affected
  • The footer will maintain its appearance across all pages
