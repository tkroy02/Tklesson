Part A: Rationalization of Surds

Question 1: \(\lim_{x \to 4} \frac{\sqrt{x} - 2}{x - 4}\)

This limit results in the indeterminate form \(\frac{0}{0}\). We solve it by rationalizing the numerator.

  1. Multiply the numerator and denominator by the conjugate of the numerator, which is \((\sqrt{x} + 2)\).
    \[ \lim_{x \to 4} \frac{\sqrt{x} - 2}{x - 4} \cdot \frac{\sqrt{x} + 2}{\sqrt{x} + 2} \]
  2. Expand the numerator using the difference of squares formula, \((a-b)(a+b) = a^2 - b^2\).
    \[ \lim_{x \to 4} \frac{(\sqrt{x})^2 - 2^2}{(x - 4)(\sqrt{x} + 2)} = \lim_{x \to 4} \frac{x - 4}{(x - 4)(\sqrt{x} + 2)} \]
  3. Cancel the common factor \((x-4)\).
    \[ \lim_{x \to 4} \frac{1}{\sqrt{x} + 2} \]
  4. Substitute \(x=4\) to find the limit.
    \[ \frac{1}{\sqrt{4} + 2} = \frac{1}{2 + 2} = \frac{1}{4} \]

Final Answer: \(\frac{1}{4}\)

Question 2: \(\lim_{x \to 9} \frac{\sqrt{x} - 3}{x - 9}\)

Similar to Question 1, we use rationalization to resolve the \(\frac{0}{0}\) indeterminate form.

  1. Multiply by the conjugate of the numerator, \((\sqrt{x} + 3)\).
    \[ \lim_{x \to 9} \frac{\sqrt{x} - 3}{x - 9} \cdot \frac{\sqrt{x} + 3}{\sqrt{x} + 3} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 9} \frac{x - 9}{(x - 9)(\sqrt{x} + 3)} \]
  3. Cancel the common factor \((x-9)\).
    \[ \lim_{x \to 9} \frac{1}{\sqrt{x} + 3} \]
  4. Substitute \(x=9\).
    \[ \frac{1}{\sqrt{9} + 3} = \frac{1}{3 + 3} = \frac{1}{6} \]

Final Answer: \(\frac{1}{6}\)

Question 3: \(\lim_{x \to 0} \frac{\sqrt{x + 1} - 1}{x}\)

This is a classic rationalization problem for the \(\frac{0}{0}\) indeterminate form.

  1. Multiply by the conjugate of the numerator, \((\sqrt{x+1} + 1)\).
    \[ \lim_{x \to 0} \frac{\sqrt{x + 1} - 1}{x} \cdot \frac{\sqrt{x + 1} + 1}{\sqrt{x + 1} + 1} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 0} \frac{(x + 1) - 1}{x(\sqrt{x + 1} + 1)} = \lim_{x \to 0} \frac{x}{x(\sqrt{x + 1} + 1)} \]
  3. Cancel the common factor \(x\).
    \[ \lim_{x \to 0} \frac{1}{\sqrt{x + 1} + 1} \]
  4. Substitute \(x=0\).
    \[ \frac{1}{\sqrt{0 + 1} + 1} = \frac{1}{1 + 1} = \frac{1}{2} \]

Final Answer: \(\frac{1}{2}\)

Question 4: \(\lim_{x \to 0} \frac{\sqrt{4 + x} - 2}{x}\)

Rationalize the numerator to handle the \(\frac{0}{0}\) form.

  1. Multiply by the conjugate, \((\sqrt{4+x} + 2)\).
    \[ \lim_{x \to 0} \frac{\sqrt{4 + x} - 2}{x} \cdot \frac{\sqrt{4 + x} + 2}{\sqrt{4 + x} + 2} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 0} \frac{(4 + x) - 4}{x(\sqrt{4 + x} + 2)} = \lim_{x \to 0} \frac{x}{x(\sqrt{4 + x} + 2)} \]
  3. Cancel the common factor \(x\).
    \[ \lim_{x \to 0} \frac{1}{\sqrt{4 + x} + 2} \]
  4. Substitute \(x=0\).
    \[ \frac{1}{\sqrt{4 + 0} + 2} = \frac{1}{2 + 2} = \frac{1}{4} \]

Final Answer: \(\frac{1}{4}\)

Question 5: \(\lim_{x \to 0} \frac{\sqrt{1 + x} - \sqrt{1 - x}}{x}\)

This problem involves two surds. We rationalize the numerator to simplify the expression.

  1. Multiply by the conjugate, \((\sqrt{1+x} + \sqrt{1-x})\).
    \[ \lim_{x \to 0} \frac{\sqrt{1 + x} - \sqrt{1 - x}}{x} \cdot \frac{\sqrt{1 + x} + \sqrt{1 - x}}{\sqrt{1 + x} + \sqrt{1 - x}} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 0} \frac{(1 + x) - (1 - x)}{x(\sqrt{1 + x} + \sqrt{1 - x})} = \lim_{x \to 0} \frac{2x}{x(\sqrt{1 + x} + \sqrt{1 - x})} \]
  3. Cancel the common factor \(x\).
    \[ \lim_{x \to 0} \frac{2}{\sqrt{1 + x} + \sqrt{1 - x}} \]
  4. Substitute \(x=0\).
    \[ \frac{2}{\sqrt{1 + 0} + \sqrt{1 - 0}} = \frac{2}{1 + 1} = \frac{2}{2} = 1 \]

Final Answer: \(1\)

Question 6: \(\lim_{x \to 1} \frac{\sqrt{x + 3} - 2}{x - 1}\)

Resolve the \(\frac{0}{0}\) indeterminate form by rationalizing the numerator.

  1. Multiply by the conjugate, \((\sqrt{x+3} + 2)\).
    \[ \lim_{x \to 1} \frac{\sqrt{x + 3} - 2}{x - 1} \cdot \frac{\sqrt{x + 3} + 2}{\sqrt{x + 3} + 2} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 1} \frac{(x + 3) - 4}{(x - 1)(\sqrt{x + 3} + 2)} = \lim_{x \to 1} \frac{x - 1}{(x - 1)(\sqrt{x + 3} + 2)} \]
  3. Cancel the common factor \((x-1)\).
    \[ \lim_{x \to 1} \frac{1}{\sqrt{x + 3} + 2} \]
  4. Substitute \(x=1\).
    \[ \frac{1}{\sqrt{1 + 3} + 2} = \frac{1}{\sqrt{4} + 2} = \frac{1}{2 + 2} = \frac{1}{4} \]

Final Answer: \(\frac{1}{4}\)

Question 7: \(\lim_{x \to 0} \frac{\sqrt{x^2 + 9} - 3}{x^2}\)

This problem uses rationalization with a variable inside the square root to handle the \(\frac{0}{0}\) form.

  1. Multiply by the conjugate, \((\sqrt{x^2+9} + 3)\).
    \[ \lim_{x \to 0} \frac{\sqrt{x^2 + 9} - 3}{x^2} \cdot \frac{\sqrt{x^2 + 9} + 3}{\sqrt{x^2 + 9} + 3} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 0} \frac{(x^2 + 9) - 9}{x^2(\sqrt{x^2 + 9} + 3)} = \lim_{x \to 0} \frac{x^2}{x^2(\sqrt{x^2 + 9} + 3)} \]
  3. Cancel the common factor \(x^2\).
    \[ \lim_{x \to 0} \frac{1}{\sqrt{x^2 + 9} + 3} \]
  4. Substitute \(x=0\).
    \[ \frac{1}{\sqrt{0 + 9} + 3} = \frac{1}{3 + 3} = \frac{1}{6} \]

Final Answer: \(\frac{1}{6}\)

Question 8: \(\lim_{x \to 0} \frac{\sqrt{1 + x} - 1 - x/2}{x^2}\)

This limit is a classic example that benefits from a **Taylor series expansion** or L'Hôpital's Rule. We will use the Taylor series for clarity.

  1. Recall the binomial series for a function of the form \((1+x)^n\) for small \(x\):
    \[ (1+x)^n = 1 + nx + \frac{n(n-1)}{2!}x^2 + \dots \]
  2. For \(\sqrt{1+x} = (1+x)^{1/2}\), we have \(n=1/2\). The approximation is:
    \[ \sqrt{1+x} \approx 1 + \frac{1}{2}x + \frac{\frac{1}{2}(\frac{1}{2}-1)}{2}x^2 = 1 + \frac{1}{2}x - \frac{1}{8}x^2 \]
  3. Substitute this approximation into the limit expression:
    \[ \lim_{x \to 0} \frac{(1 + \frac{1}{2}x - \frac{1}{8}x^2) - 1 - \frac{1}{2}x}{x^2} \]
  4. Simplify the numerator. The \(1\) and \(\frac{1}{2}x\) terms cancel out, leaving only the \(x^2\) term.
    \[ \lim_{x \to 0} \frac{-\frac{1}{8}x^2}{x^2} = -\frac{1}{8} \]

Final Answer: \(-\frac{1}{8}\)

Question 9: \(\lim_{x \to 4} \frac{\sqrt{5x + 4} - 6}{x - 4}\)

The original problem had a typo in the numerator that led to a non-\(\frac{0}{0}\) form. The correct version is likely \(\lim_{x \to 4} \frac{\sqrt{5x+16}-6}{x-4}\), which gives the \(\frac{0}{0}\) form. Let's solve that version.

  1. Multiply by the conjugate, \((\sqrt{5x+16} + 6)\).
    \[ \lim_{x \to 4} \frac{\sqrt{5x + 16} - 6}{x - 4} \cdot \frac{\sqrt{5x + 16} + 6}{\sqrt{5x + 16} + 6} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 4} \frac{(5x + 16) - 36}{(x - 4)(\sqrt{5x + 16} + 6)} = \lim_{x \to 4} \frac{5x - 20}{(x - 4)(\sqrt{5x + 16} + 6)} \]
  3. Factor out a 5 from the numerator and cancel the common factor \((x-4)\).
    \[ \lim_{x \to 4} \frac{5(x - 4)}{(x - 4)(\sqrt{5x + 16} + 6)} = \lim_{x \to 4} \frac{5}{\sqrt{5x + 16} + 6} \]
  4. Substitute \(x=4\).
    \[ \frac{5}{\sqrt{5(4) + 16} + 6} = \frac{5}{\sqrt{36} + 6} = \frac{5}{6 + 6} = \frac{5}{12} \]

Final Answer: \(\frac{5}{12}\)

Question 10: \(\lim_{x \to 0} \frac{\sqrt{2 + x} - \sqrt{2 - x}}{x}\)

This problem is a rationalization of a difference of two surds, yielding a \(\frac{0}{0}\) form.

  1. Multiply by the conjugate, \((\sqrt{2+x} + \sqrt{2-x})\).
    \[ \lim_{x \to 0} \frac{\sqrt{2 + x} - \sqrt{2 - x}}{x} \cdot \frac{\sqrt{2 + x} + \sqrt{2 - x}}{\sqrt{2 + x} + \sqrt{2 - x}} \]
  2. Simplify the numerator.
    \[ \lim_{x \to 0} \frac{(2 + x) - (2 - x)}{x(\sqrt{2 + x} + \sqrt{2 - x})} = \lim_{x \to 0} \frac{2x}{x(\sqrt{2 + x} + \sqrt{2 - x})} \]
  3. Cancel the common factor \(x\).
    \[ \lim_{x \to 0} \frac{2}{\sqrt{2 + x} + \sqrt{2 - x}} \]
  4. Substitute \(x=0\).
    \[ \frac{2}{\sqrt{2 + 0} + \sqrt{2 - 0}} = \frac{2}{\sqrt{2} + \sqrt{2}} = \frac{2}{2\sqrt{2}} = \frac{1}{\sqrt{2}} \]

Final Answer: \(\frac{1}{\sqrt{2}}\)


Part B: Trigonometry in Limits

Question 11: \(\lim_{x \to 0} \frac{\sin(x)}{x}\)

This is a fundamental and well-known special trigonometric limit. It is proven using the Squeeze Theorem and is a key concept in calculus. You should memorize this result.

Final Answer: \(1\)

Question 12: \(\lim_{x \to 0} \frac{\tan(x)}{x}\)

This limit results in the \(\frac{0}{0}\) indeterminate form. We can rewrite it using the identity \(\tan(x) = \frac{\sin(x)}{\cos(x)}\).

  1. Rewrite the expression.
    \[ \lim_{x \to 0} \frac{\frac{\sin(x)}{\cos(x)}}{x} = \lim_{x \to 0} \frac{\sin(x)}{x \cos(x)} \]
  2. Separate the limit into two parts.
    \[ \lim_{x \to 0} \frac{\sin(x)}{x} \cdot \lim_{x \to 0} \frac{1}{\cos(x)} \]
  3. Apply the known limit \(\lim_{x \to 0} \frac{\sin(x)}{x} = 1\) and substitute \(x=0\) into the second part.
    \[ (1) \cdot \frac{1}{\cos(0)} = 1 \cdot \frac{1}{1} = 1 \]

Final Answer: \(1\)

Question 13: \(\lim_{x \to 0} \frac{\sin(5x)}{x}\)

This is a variation of the fundamental trigonometric limit. We need to create the form \(\frac{\sin(u)}{u}\).

  1. Multiply the numerator and denominator by 5.
    \[ \lim_{x \to 0} \frac{5 \sin(5x)}{5x} \]
  2. Pull the constant 5 out of the limit. Let \(u = 5x\). As \(x \to 0\), \(u \to 0\).
    \[ 5 \lim_{u \to 0} \frac{\sin(u)}{u} \]
  3. Apply the fundamental limit.
    \[ 5 \cdot 1 = 5 \]

Final Answer: \(5\)

Question 14: \(\lim_{x \to 0} \frac{\sin(3x)}{\sin(2x)}\)

To solve this, we will use the trick of dividing both the numerator and denominator by \(x\).

  1. Divide the numerator and denominator by \(x\).
    \[ \lim_{x \to 0} \frac{\frac{\sin(3x)}{x}}{\frac{\sin(2x)}{x}} \]
  2. Apply the limit to the numerator and denominator separately. Using the general rule \(\lim_{x \to 0} \frac{\sin(kx)}{x} = k\).
    \[ \frac{\lim_{x \to 0} \frac{\sin(3x)}{x}}{\lim_{x \to 0} \frac{\sin(2x)}{x}} = \frac{3}{2} \]

Final Answer: \(\frac{3}{2}\)

Question 15: \(\lim_{x \to 0} \frac{1 - \cos(x)}{x}\)

This is another fundamental special trigonometric limit that results in \(\frac{0}{0}\).

  1. Multiply the numerator and denominator by the conjugate of the numerator, \((1+\cos(x))\).
    \[ \lim_{x \to 0} \frac{1 - \cos(x)}{x} \cdot \frac{1 + \cos(x)}{1 + \cos(x)} \]
  2. Use the Pythagorean identity \(1-\cos^2(x) = \sin^2(x)\).
    \[ \lim_{x \to 0} \frac{1 - \cos^2(x)}{x(1 + \cos(x))} = \lim_{x \to 0} \frac{\sin^2(x)}{x(1 + \cos(x))} \]
  3. Rewrite the expression to use the fundamental limit.
    \[ \lim_{x \to 0} \frac{\sin(x)}{x} \cdot \frac{\sin(x)}{1 + \cos(x)} \]
  4. Apply the limit to each part.
    \[ \left( \lim_{x \to 0} \frac{\sin(x)}{x} \right) \cdot \left( \lim_{x \to 0} \frac{\sin(x)}{1 + \cos(x)} \right) = (1) \cdot \frac{\sin(0)}{1 + \cos(0)} = 1 \cdot \frac{0}{1+1} = 0 \]

Final Answer: \(0\)

Question 16: \(\lim_{x \to 0} \frac{1 - \cos(x)}{x^2}\)

This is also a fundamental limit. We can use the same technique as the previous problem.

  1. Multiply by the conjugate \((1+\cos(x))\).
    \[ \lim_{x \to 0} \frac{1 - \cos(x)}{x^2} \cdot \frac{1 + \cos(x)}{1 + \cos(x)} \]
  2. Use the identity \(1-\cos^2(x) = \sin^2(x)\).
    \[ \lim_{x \to 0} \frac{\sin^2(x)}{x^2(1 + \cos(x))} \]
  3. Rewrite the expression.
    \[ \lim_{x \to 0} \left( \frac{\sin(x)}{x} \right)^2 \cdot \frac{1}{1 + \cos(x)} \]
  4. Apply the limit to each part.
    \[ \left( \lim_{x \to 0} \frac{\sin(x)}{x} \right)^2 \cdot \lim_{x \to 0} \frac{1}{1 + \cos(x)} = (1)^2 \cdot \frac{1}{1 + \cos(0)} = 1 \cdot \frac{1}{1+1} = \frac{1}{2} \]

Final Answer: \(\frac{1}{2}\)

Question 17: \(\lim_{x \to 0} \frac{\sin(x)}{1 - \cos(x)}\)

This limit evaluates to the indeterminate form \(\frac{0}{0}\). We can resolve this using the half-angle identities.

  1. Use the half-angle identities: \(\sin(x) = 2\sin(\frac{x}{2})\cos(\frac{x}{2})\) and \(1-\cos(x) = 2\sin^2(\frac{x}{2})\).
    \[ \lim_{x \to 0} \frac{2\sin(\frac{x}{2})\cos(\frac{x}{2})}{2\sin^2(\frac{x}{2})} \]
  2. Cancel the common terms.
    \[ \lim_{x \to 0} \frac{\cos(\frac{x}{2})}{\sin(\frac{x}{2})} = \lim_{x \to 0} \cot(\frac{x}{2}) \]
  3. As \(x \to 0\), the expression approaches \(\cot(0)\), which is undefined. The limit from the right (\(x \to 0^+\)) is \(+\infty\) and the limit from the left (\(x \to 0^-\)) is \(-\infty\).

Final Answer: Diverges (does not exist)

Question 18: \(\lim_{x \to 0} \frac{\tan(4x)}{\sin(2x)}\)

We use the same technique as Question 14, dividing both the numerator and denominator by \(x\) to utilize the fundamental limits.

  1. Divide the numerator and denominator by \(x\).
    \[ \lim_{x \to 0} \frac{\frac{\tan(4x)}{x}}{\frac{\sin(2x)}{x}} \]
  2. Apply the general trigonometric limits: \(\lim_{x \to 0} \frac{\tan(kx)}{x} = k\) and \(\lim_{x \to 0} \frac{\sin(kx)}{x} = k\).
    \[ \frac{\lim_{x \to 0} \frac{\tan(4x)}{x}}{\lim_{x \to 0} \frac{\sin(2x)}{x}} = \frac{4}{2} = 2 \]

Final Answer: \(2\)

Question 19: \(\lim_{x \to 0} \frac{\sin(2x) - 2x}{x^3}\)

This limit results in the indeterminate form \(\frac{0}{0}\). A powerful way to solve this is by using the **Taylor series expansion** for \(\sin(x)\).

  1. Recall the Maclaurin series for \(\sin(y)\):
    \[ \sin(y) = y - \frac{y^3}{3!} + \frac{y^5}{5!} - \dots \]
  2. Substitute \(y = 2x\) into the series.
    \[ \sin(2x) = 2x - \frac{(2x)^3}{3!} + \dots = 2x - \frac{8x^3}{6} + \dots \]
  3. Substitute this back into the limit expression.
    \[ \lim_{x \to 0} \frac{(2x - \frac{8x^3}{6} + \dots) - 2x}{x^3} = \lim_{x \to 0} \frac{-\frac{8x^3}{6} + \text{higher order terms}}{x^3} \]
  4. Cancel the common factor \(x^3\).
    \[ \lim_{x \to 0} (-\frac{8}{6} + \dots) = -\frac{4}{3} \]

Final Answer: \(-\frac{4}{3}\)

Question 20: \(\lim_{x \to 0} \frac{\tan^2(x)}{x^2}\)

This problem can be solved by rewriting the expression using exponent properties.

  1. Rewrite the expression as a squared term.
    \[ \lim_{x \to 0} \left( \frac{\tan(x)}{x} \right)^2 \]
  2. Using the limit property \(\lim_{x \to c} [f(x)]^n = [\lim_{x \to c} f(x)]^n\), we apply the limit to the inner function.
    \[ \left( \lim_{x \to 0} \frac{\tan(x)}{x} \right)^2 \]
  3. Apply the known fundamental limit, \(\lim_{x \to 0} \frac{\tan(x)}{x} = 1\).
    \[ (1)^2 = 1 \]

Final Answer: \(1\)
