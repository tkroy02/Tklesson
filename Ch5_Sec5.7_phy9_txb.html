5.7 Mechanical Advantage

Mechanical advantage (MA) is a measure of how much a machine multiplies the input force. It shows how “easy” a machine makes work by allowing less input force over a greater distance to achieve the same output.

Key Concept: Machines trade off force and distance: if the input force is small, it must move over a longer distance, and vice versa.

Formulas for Mechanical Advantage

Mechanical advantage can be calculated using forces:

\[ MA = \frac{F_\text{out}}{F_\text{in}} \]

Or using distances moved by input and output forces:

\[ MA = \frac{d_\text{in}}{d_\text{out}} \]

Example: If the input-to-output force ratio is 1:4, the distance ratio is 4:1, indicating a trade-off between force and distance.

When \( F_\text{in} > F_\text{out} \), the machine has a mechanical disadvantage.

Lever Classes

Levers are simple machines that illustrate mechanical advantage. There are three classes:

  • First-class lever: Fulcrum is between input and output forces (e.g., seesaw). Can multiply force or distance depending on arm lengths.
  • Second-class lever: Output force is between fulcrum and input force (e.g., wheelbarrow). Always provides mechanical advantage (\( MA>1 \)).
  • Third-class lever: Input force is between fulcrum and output force (e.g., broom). Provides mechanical disadvantage (\( MA<1 \)), but increases distance/speed.

First-Class Lever

In a first-class lever, the fulcrum is placed between the input force and the output force. Depending on the lengths of the arms, the lever can either multiply force or distance.

Fulcrum between input and output forces.

Example: Seesaw, crowbar, claw-hammer.

Mechanical advantage depends on the ratio of the lengths of the input and output arms: \[ MA = \frac{L_\text{in}}{L_\text{out}} \]

Second-Class Lever

In a second-class lever, the output force is placed between the fulcrum and the input force. These levers always provide a mechanical advantage (\( MA > 1 \)), since the input arm is always longer than the output arm.

Output force between fulcrum and input force.

Example: Wheelbarrow, nutcracker.

Third-Class Lever

In a third-class lever, the input force is placed between the fulcrum and the output force. These levers provide a mechanical disadvantage (\( MA < 1 \)), but increase the distance or speed of the output motion.

Input force between fulcrum and output force.

Example: Broom, human arm (lifting a weight with the hand), sugar tong, fishing rod.

Worked Example

A lever lifts a 200 N load using an input force of 50 N. Calculate the mechanical advantage.

Solution:
\[ MA = \frac{F_\text{out}}{F_\text{in}} = \frac{200}{50} = 4 \]
Answer: The lever has a mechanical advantage of 4.

Practice Problems

  1. A pulley system requires an input force of 100 N to lift a 500 N load. Determine the mechanical advantage.
  2. A ramp raises a crate 2 m using a ramp 10 m long. Calculate the ideal mechanical advantage (ignore friction).
  3. An inclined plane allows a 150 N box to be pushed up using 50 N. Find the mechanical advantage and indicate whether it is advantageous.

Sample Answers

  1. \( MA = \frac{500}{100} = 5 \)
  2. \( MA_\text{ideal} = \frac{d_\text{in}}{d_\text{out}} = \frac{10}{2} = 5 \)
  3. \( MA = \frac{150}{50} = 3 \), mechanical advantage achieved.

Summary

  • Mechanical advantage shows how a machine multiplies or trades off input force and distance.
  • MA can be calculated using forces or distances.
  • Real machines have friction, so actual MA is less than ideal MA.
  • Force-distance trade-off: smaller input force requires greater input distance, and vice versa.
  • Levers demonstrate different types of MA depending on their class (first, second, third).
