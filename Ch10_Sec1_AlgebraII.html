Section 10.1: Data Analysis & Displays

This section introduces methods for organizing and displaying data, including frequency tables, histograms, bar graphs, and stem-and-leaf plots.

Example 1: Frequency Table

Construct a frequency table for the data: 2, 3, 5, 3, 4, 2, 5, 3.

Count occurrences of each value:

  • 2 → 2 times
  • 3 → 3 times
  • 4 → 1 time
  • 5 → 2 times

Frequency table completed.

Example 2: Bar Graph

Using the previous frequency table, draw a bar graph representing the data.

Each value on the x-axis: 2, 3, 4, 5.

Height of bars corresponds to frequency: 2, 3, 1, 2.

Graph visualizes frequency distribution.

Practice Problems

  1. Data: 1, 2, 2, 3, 3, 3, 4. Construct a frequency table.
  2. Draw a histogram for the data: 5, 7, 5, 6, 8, 7, 5, 6.
