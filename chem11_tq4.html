Introduction to Acids and Bases — Identification, Properties & Litmus Tests

🧪 Acids and Bases Challenge

🔴 ACIDS
✓ Produce H⁺ / H₃O⁺ ions
✓ Taste sour
✓ Turn blue litmus → red
✓ React with metals → H₂ gas
✓ Examples: HCl, H₂SO₄, CH₃COOH
🔵 BASES
✓ Produce OH⁻ ions
✓ Taste bitter
✓ Turn red litmus → blue
✓ Feel slippery
✓ Examples: NaOH, KOH, Ca(OH)₂
Blue Litmus + Acid → 🔴 RED Red Litmus + Base → 🔵 BLUE

This 10-question MCQ quiz tests your ability to identify acids and bases, recall their key properties, and understand the Arrhenius definition.

🔬 Skills Tested:

  • ✓ Identifying common acids and bases by formula
  • ✓ Understanding Arrhenius definitions (H⁺ vs. OH⁻ production)
  • ✓ Knowing the litmus test results for acids and bases
  • ✓ Recalling that acids react with metals to produce H₂ gas
  • ✓ Recognizing acid taste (sour) vs. base taste (bitter)
  • ✓ Identifying hydronium ion (H₃O⁺) formation in water
