Section 11.4: Trig Identities & Equations

This section focuses on verifying trig identities and solving basic trig equations using fundamental identities.

Example 1: Verifying an Identity

Verify \( \sin^2 x + \cos^2 x = 1 \).

Step 1: Recall the Pythagorean identity \( \sin^2 x + \cos^2 x = 1 \)

Step 2: The left-hand side is exactly \( \sin^2 x + \cos^2 x \)

Step 3: Therefore, the identity is verified

Example 2: Solving a Trig Equation

Solve \( 2\sin x - 1 = 0 \) for \( 0 \le x < 2\pi \).

Step 1: Isolate sin: \( 2\sin x = 1 \Rightarrow \sin x = 1/2 \)

Step 2: Find all solutions in the given interval: \( x = \pi/6, 5\pi/6 \)

Practice Problems

  1. Verify \( 1 + \tan^2 x = \sec^2 x \)
  2. Solve \( \cos x = -1/2 \) for \( 0 \le x < 2\pi \)
  3. Verify \( \sin 2x = 2\sin x \cos x \)
  4. Solve \( 1 - 2\cos^2 x = 0 \) for \( 0 \le x < 2\pi \)
  5. Verify \( 1 - \sin^2 x = \cos^2 x \)
