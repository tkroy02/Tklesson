Section 9.7: Convex Mirrors

A convex mirror is a spherical mirror that curves outward. It diverges light rays and always produces virtual, upright, and diminished images.

Mirror Formula

The same mirror formula applies:

\[ \frac{1}{f} = \frac{1}{u} + \frac{1}{v} \]

Sign conventions for convex mirrors:

  • Focal length (\(f\)) is negative.
  • Image distance (\(v\)) is negative (virtual images behind the mirror).
  • Object distance (\(u\)) is positive.

Example 1: Object in Front of Convex Mirror

Problem: An object is placed 15 cm from a convex mirror with focal length 20 cm. Find the image distance.

Given:

  • \(u = 15 \, \mathrm{cm}\)
  • \(f = -20 \, \mathrm{cm}\) (convex mirror)

Solution:

\[ \frac{1}{v} = \frac{1}{f} - \frac{1}{u} = \frac{1}{-20} - \frac{1}{15} = -\frac{1}{20} - \frac{1}{15} = -\frac{7}{60} \] \[ v = -\frac{60}{7} \approx -8.57 \, \mathrm{cm} \]

Answer: The image forms 8.57 cm behind the mirror, virtual and upright.

Example 2: Object Far from Convex Mirror

Problem: An object is 100 cm from a convex mirror with \(f = -25 \, \mathrm{cm}\). Find the image distance and magnification.

Given:

  • \(u = 100 \, \mathrm{cm}\)
  • \(f = -25 \, \mathrm{cm}\)

Solution:

\[ \frac{1}{v} = \frac{1}{f} - \frac{1}{u} = \frac{1}{-25} - \frac{1}{100} = -\frac{1}{25} - \frac{1}{100} = -\frac{5}{100} = -\frac{1}{20} \] \[ v = -20 \, \mathrm{cm} \]

Magnification \(m = -\frac{v}{u} = -\frac{-20}{100} = 0.2\)

Answer: Image forms 20 cm behind mirror, virtual, upright, and 0.2 times smaller than the object.

Key Points

  • Convex mirrors always produce virtual, upright, and diminished images.
  • Focal length is negative; image distance is negative.
  • Useful for rear-view mirrors, surveillance, and wide-angle reflections.
  • Magnification \(m = -\frac{v}{u}\) and is always less than 1 for convex mirrors.
